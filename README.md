# De-Boor Algorithm:
```
Input: k                     // Spline Degree, which is also required multiplicity
       P=[P_0,P_1,...,P_N-1] // Control Point 
       t*                    // Parameter of interest
Output: P(t*)                // Evaluation of Spline at t*
       
```
